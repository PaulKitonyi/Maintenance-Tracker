This is a Maintenance Tracker Application whereby you can log in and make requests concerning your issues whether maintenance or repair.

Features

  • As a User you can register and Login.
  • As a logged in User you are able to make maintenance or repairs request.
  • As a logged in User you can view all your requests.

Maintenance Tracker

  • Maintenance Tracker is a web portal whereby you can make request concerning maintenance and repairs.
  • An Admin can approve or disapprove your requests and resolve them.
  • To make it better an Admin can give you feedback because we strongly believe that we thrive on it.
